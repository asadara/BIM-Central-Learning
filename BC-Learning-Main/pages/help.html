Panduan Membuat Akun BCL

Ikuti langkah berikut jika Anda baru bergabung dan belum memiliki akun pengguna BCL.

1Buka alamat resmi BCL

Gunakan https://bcl.nke.net. Jika muncul peringatan sertifikat, ikuti panduan instalasi Root CA pada bagian HTTPS di halaman ini.

2Buka halaman pendaftaran

Pilih menu Akun pada navbar, lalu pilih Daftar. Anda juga dapat membuka tombol Buat Akun di bawah panduan ini.

3Pilih metode pendaftaran

Gunakan formulir akun BCL atau tombol Google. Untuk pendaftaran manual, siapkan username, email aktif, password, dan BIM Experience Level.

4Lengkapi data pekerjaan

Isi Job Role dan Organization agar admin dapat mengenali unit kerja dan kebutuhan akses Anda. Kedua field ini opsional, tetapi sangat disarankan.

5Selesaikan registrasi

Pilih Create Account atau selesaikan proses Google. Akun akan aktif setelah registrasi berhasil; saat ini tidak ada langkah verifikasi email terpisah.

6Login dan lengkapi profil

Masuk melalui menu Akun > Login, lalu periksa profil, BIM Level, role pekerjaan, dan organisasi Anda sebelum menggunakan fitur BCL.

Catatan akses: pembuatan akun tidak otomatis memberikan akses ke seluruh menu internal. Beberapa halaman memerlukan persetujuan dan pengaturan permission oleh admin BCL.

Mengatasi Peringatan HTTPS Merah

Jika browser menampilkan peringatan saat membuka https://bcl.nke.net, pilih panduan sesuai perangkat Anda.

Windows / Desktop
  1. Unduh installer Root CA LAN `cert.bat`.
  2. Klik kanan file tersebut lalu pilih Run as administrator.
  3. Jika muncul SmartScreen, pilih More info lalu Run anyway.
  4. Setelah selesai, tutup dan buka ulang browser.
Download cert.bat Download Root CA (.crt)
Android / Mobile
  1. Unduh file Root CA (.crt) ke perangkat Android Anda.
  2. Buka Settings lalu cari menu seperti Security, Encryption & credentials, atau Install a certificate.
  3. Pilih CA certificate, lalu pilih file bcl-lan-root-ca.crt yang sudah diunduh.
  4. Tutup lalu buka ulang Chrome/browser setelah import selesai.
Download Root CA (.crt)
Installer Windows juga akan menambahkan hosts mapping 10.0.0.90 bcl.nke.net agar akses via WiFi/WLAN berjalan.
Untuk Android, file yang dipakai tetap Root CA (.crt). Android tidak menggunakan file .bat.
Catatan: Firefox memakai store sertifikat sendiri. Di Android dan desktop, peringatan tetap bisa muncul jika akses memakai IP atau hostname yang tidak sesuai sertifikat. Gunakan https://bcl.nke.net setelah trust certificate terpasang.

FAQ - Pertanyaan Umum

Bagaimana cara mendaftar kursus gratis?
Kursus gratis dapat diakses langsung tanpa pendaftaran khusus. Cukup login ke akun Anda dan klik "Mulai" pada kursus yang diinginkan.
Apakah saya mendapatkan sertifikat setelah menyelesaikan kursus?
Ya, Anda akan mendapatkan sertifikat digital setelah menyelesaikan kursus dengan nilai minimal 70%. Sertifikat dapat diunduh dari dashboard Anda.
Bagaimana cara mengakses BIM NKE?
BIM NKE dapat diakses melalui menu E-Learning > BIM NKE. Platform ini berisi knowledge base, artikel, dan referensi BIM terstruktur.
Apakah ada biaya untuk menggunakan platform ini?
Platform BCL menyediakan banyak konten gratis. Kursus premium dan sertifikasi tertentu mungkin memerlukan biaya, tetapi akan diinformasikan dengan jelas.
Mengapa menu tertentu tidak terlihat setelah saya membuat akun?
Beberapa menu internal memakai permission khusus dan tidak aktif otomatis saat registrasi. Ajukan akses melalui Akun > Pesan Masuk atau email ke nke-bim@nusakonstruksi.com. Setelah disetujui admin, muat ulang halaman atau login kembali.
Bagaimana cara mengikuti forum diskusi?
Akses forum melalui menu Community > Forum Diskusi. Anda dapat membuat topik baru atau berkontribusi pada diskusi yang ada setelah login.
