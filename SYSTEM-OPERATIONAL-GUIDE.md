## 🔧 TROUBLESHOOTING QUICK FIXES

### SSL Certificate Warning
```
Issue: Browser shows "Not Secure"
Solution: Click "Advanced" → "Proceed to localhost (unsafe)"
Alternative: Use HTTP version at http://localhost:5151
```

### Port Already in Use
```
Check: netstat -an | findstr :5150
Kill: taskkill /PID [PID_NUMBER] /F
```

### Module Not Found
```
Fix: cd C:\BCL\backend && npm install
```

### Permission Denied
```
Fix: Run PowerShell as Administrator
```
